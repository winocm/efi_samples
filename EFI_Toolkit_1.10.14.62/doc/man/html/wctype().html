NAME

     wctype - Determines a classification rule for wide-character codes.

SYNOPSIS

     #include <wctype.h>

     wctype_t
     wctype(const wchar_t * property)

DESCRIPTION

     wctype determines a classification rule for wide-character codes.
     The following pairs of calls have the same behavior in all locales
     (but an implementation can define additional classification rules 
     even in the "C" locale):

     iswalnum(c) same as  iswctype(c, wctype("alnum"))

     iswalpha(c) same as  iswctype(c, wctype("alpha"))

     iswcntrl(c) same as  iswctype(c, wctype("cntrl"))

     iswdigit(c) same as  iswctype(c, wctype("digit"))

     iswgraph(c) same as  iswctype(c, wctype("graph"))

     iswlower(c) same as  iswctype(c, wctype("lower"))

     iswprint(c) same as  iswctype(c, wctype("print"))

     iswpunct(c) same as  iswctype(c, wctype("punct"))

     iswspace(c) same as  iswctype(c, wctype("space"))

     iswupper(c) same as  iswctype(c, wctype("upper"))

     iswxdigit(c) same as  iswctype(c, wctype("xdigit"))

RETURN VALUES

     If the LC_CTYPE category of the current locale does not define 
     a classification rule whose name matches the property string 
     property, The wctype() functions returns zero. Otherwise, it 
     returns a nonzero value suitable for use as the second argument 
     to a subsequent call to towctrans.

SEE ALSO

     iswctype(3)

STANDARDS

     The wctype() function conforms to ISO 9899: 1990 (``ISO C'').
