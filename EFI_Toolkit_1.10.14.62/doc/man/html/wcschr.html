NAME

     wcschr - locate wide character in string

SYNOPSIS

     #include <string.h>

     wchar_t *
     wcschr(const wchar_t *s, int c)

DESCRIPTION

     The wcschr() function locates the first occurrence of c in the string
     pointed to by s. The terminating NUL character is considered part of the
     string.  If c is `\0', wcschr() locates the terminating `\0'.

RETURN VALUES

     The function wcschr() returns a pointer to the located character, or NULL
     if the character does not appear in the string.
