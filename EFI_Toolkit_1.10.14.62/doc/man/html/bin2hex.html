NAME

     bin2hex - Give binary file a hexadecimal readable format

SYNOPSIS

     bin2hex Inputfile

DESCRIPTION

     The bin2hex utility is a Win32 application.  It reads an 
     Inputfile and produces a corresponding output file which represents
     the original input file byte by byte in readable hexadecimal
     format. The output file's name is the input file's name plus the
     extension '.txt'.          


     Options list: 

     Inputfile : the name of the input file ( only one input 
                        file is allowed )

     The bin2hex utility is mainly used with the following files as
     input: LC_CTYPE, LC_COLLATE and LC_TIME. 
     
     For each file, bin2hex produces a block of text in hexadecimal
     format which can be inserted into the related C source file which
     implements a certain locale. The inserted part serves as global variable 
     initialization data and can be handled by the C compiler to allocate 
     storage in the resulting '.efi' image. In such a way, there is no need
     to handle disk files, since the data are available within the program.
