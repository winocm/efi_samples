NAME

     catclose - close message catalog

SYNOPSIS

     #include <nl_types.h>

     int
     catclose(nl_catd catd)

DESCRIPTION

     The catclose() function closes the message catalog specified by the argu-
     ment catd.

SEE ALSO

     catgets(3),  catopen(3)

STANDARDS

     The catclose() function conforms to X/Open Portability Guide Issue 4
     (``XPG4'').
