NAME

     mkramdisk - create and map RAM disks
     ramdisk.efi - default RAM disk driver

SYNOPSIS

     mkramdisk [-s size] [-d driver] drive_mapping ....

DESCRIPTION

     Mkramdisk is used to load and configure one or more instances of a RAM disk
     driver.  The optional -s and -d switches may be repeated and will apply to
     subsequent drive_mapping instances.

     The following options are available:

     -s      Sets the size of the RAM disk in megabytes.  The default size is
             32MB.

     -d      Sets the name of the EFI RAM disk driver to be loaded.  The default
             file is ramdisk.efi.  If an explicit or relative path is not
             specified with the file name, mkramdisk will search the directories
             in the path variable.

     drive_mapping
	     Specifies the EFI Shell file mapping for the disk.  The name should
	     be specified without a trailing colon.  Mkramdisk will load and
	     configure a drive for each mapping specified.  The driver and disk
	     size will be set to the defaults or to the last options value
	     specified.

NOTES

     In order to map drives, mkramdisk should be run from the EFI shell.  If run
     from the boot manager, mkramdisk will load and configure the drive but the
     EFI shell, when run, will provide its own mapping for the drive.  Likewise,
     the EFI Shell "map -r" command will remap all drives including any loaded
     RAM drives.

DIAGNOSTICS

     Messages indicating the driver(s) could not be loaded, started, or mapped.
