NAME

     wcsrchr - locate character in wide string

SYNOPSIS

     #include <string.h>

     wchar_t *
     wcsrchr(const wchar_t *s, int c)

DESCRIPTION

     The wcsrchr() function locates the last occurrence of c (converted to a
     wchar_t) in the string s. If c is `\0', wcsrchr() locates the terminating
     `\0'.

RETURN VALUES

     The wcsrchr() function returns a pointer to the character, or a null
     pointer if c does not occur anywhere in s.
