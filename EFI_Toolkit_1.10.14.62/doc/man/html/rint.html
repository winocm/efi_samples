NAME

     rint, rintf - round to integral value in floating-point format

SYNOPSIS

     #include <math.h>

     double
     rint(double x)

     float
     rintf(float x)

DESCRIPTION

     The rint() and the rintf() functions return the integral value (repre-
     sented as a double or float precision number) nearest to x according to
     the prevailing rounding mode.

SEE ALSO

     abs(3),  ceil(3),	fabs(3),  floor(3),  ieee(3),  math(3)

HISTORY

     A rint() function appeared in Version 6 AT&T UNIX.
