_LIBC_GetOpenFileDevSpecific 	  Libc    Manual	 _LIBC_GetOpenFileDevSpecific

NAME

     _LIBC_GetOpenFileDevSpecific - retrieve device specific information associated with a file
                                    descriptor

SYNOPSIS

     #include <atk_libc.h>

     INT32
     _LIBC_GetOpenFileDevSpecific(INT32 fd, VOID **DevSpecific);

DESCRIPTION

     The _LIBC_GetOpenFileDevSpecific() function returns a pointer to the device specific
     information associated with the file descriptor fd.  The device specific information
     is returned in DevSpecific.

RETURN VALUES

     0       Success

     -1      Failure
