NAME

     mklocale - convert locale source definition

SYNOPSIS

     mklocale CTYPEInputfile COLLATEInputfile TIMEInputfile

DESCRIPTION

     The mklocale utility is a Win32 application.  mklocale converts locale
     source definitions into a format usable by the locale-dependent func-
     tions.  It's the convienient away of generation all locale needed re-
     source files.  It calls mkctype, mkcollate, mktime respectively and gen-
     erates 3 binary resource files: LC_CTYPE, LC_COLLATE, LC_TIME.

FILES

     /etc/locale/<language>/LC_CTYPE      
     /etc/locale/<language>/LC_COLLATE
     /etc/locale/<language>/LC_TIME
