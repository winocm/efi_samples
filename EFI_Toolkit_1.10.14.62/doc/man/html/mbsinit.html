NAME

     mbsinit - initializes multibyte character shift state

SYNOPSIS

     #include <wchar.h>

     int
     mbsinit(const mbstate_t * pst)

DESCRIPTION

     If ps is not a null pointer, the mbsinit function determines whether the
     pointed-to mbstate_t object describes an initial conversion state.

RETURN VALUES

     The mbsinit() function returns nonzero if ps is a null pointer or if the
     pointed-to object describes an initial conversion state; otherwise, it
     returns zero.
     This function has not been actually implemented. So it will always return 0.

SEE ALSO

     btowc(3),  wctob(3),  mbrlen(3),  mbrtowc(3),  wcrtomb(3),

STANDARDS

     The mbsinit() function conforms to ISO 9899: 1990 (``ISO C'').
