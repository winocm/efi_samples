NAME

     wcscoll - compare strings according to current collation

SYNOPSIS

     #include <wchar.h>

     int
     wcscoll(const wchar_t *s1, const wchar_t *s2)

DESCRIPTION

     The wcscoll() function lexicographically compares the null-terminated
     strings s1 and s2 according to the current locale collation if any, oth-
     erwise call strcmp, and returns an integer greater than, equal to, or
     less than 0, according as s1 is greater than, equal to, or less than s2.

SEE ALSO

     setlocale(3),  wcscmp(3),  wcsxfrm(3)

STANDARDS

     The wcscoll() function conforms to ISO 9899: 1990 (``ISO C'').
