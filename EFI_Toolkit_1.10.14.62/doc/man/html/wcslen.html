NAME

     wcslen - find length of wide string

SYNOPSIS

     #include <string.h>

     size_t
     wcslen(const wchar_t *s)

DESCRIPTION

     The wcslen() function computes the length of the string s.

RETURN VALUES

     The wcslen() function returns the number of characters that precede the
     terminating NUL character.
