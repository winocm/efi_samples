NAME

     wsccspn - span the complement of a wide string

SYNOPSIS

     #include <string.h>

     size_t
     wsccspn(const wchar_t *s, const wchar_t charset)

DESCRIPTION

     The wsccspn() function spans the initial part of the null-terminated
     string s as long as the characters from s do not occur in string charset
     (it spans the complement of charset).

RETURN VALUES

     The wsccspn() function returns the number of characters spanned.
