NAME

     wcspbrk - locate multiple characters in wide string

SYNOPSIS

     #include <string.h>

     wchar_t *
     wcspbrk(const wchar_t *s, const wchar_t *charset)

DESCRIPTION

     The wcspbrk() function locates in the null-terminated string s the first
     occurrence of any character in the string charset and returns a pointer
     to this character.  If no characters from charset occur anywhere in s
     wcspbrk() returns NULL.
