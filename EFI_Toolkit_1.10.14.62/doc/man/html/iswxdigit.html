NAME

     iswxdigit - hexadecimal-digit wide character test

SYNOPSIS

     #include <wctype.h>

     int
     iswxdigit(wint_t wc)

DESCRIPTION

     The iswxdigit() function tests for any hexadecimal-digit wide character.  In
     the ASCII character set, this includes the following characters:

     060 ``0''	   061 ``1''	 062 ``2''     063 ``3''     064 ``4''
     065 ``5''	   066 ``6''	 067 ``7''     070 ``8''     071 ``9''
     101 ``A''	   102 ``B''	 103 ``C''     104 ``D''     105 ``E''
     106 ``F''	   141 ``a''	 142 ``b''     143 ``c''     144 ``d''
     145 ``e''	   146 ``f''

RETURN VALUES

     The iswxdigit() function returns zero if the wide character tests false and re-
     turns non-zero if the wide character tests true.
