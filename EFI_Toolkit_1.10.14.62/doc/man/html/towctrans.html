NAME

     towctrans - Transforms a wide character.

SYNOPSIS

     #include <wctype.h>

     wint_t 
     towctrans(wint_t c, wctrans_t category)

DESCRIPTION

     towctrans determines a mapping from one set of wide-character codes to another. 

     The following pairs of calls have the same behavior in all locales (but an 
     implementation can define additional mappings even in the "C" locale): 

     tolower( c ) same as   towctrans( c, wctrans("towlower" ) )
     towupper( c ) same as  towctrans( c, wctrans( "toupper" ) )

RETURN VALUES

     If the LC_CTYPE category of the current locale does not define a mapping 
     whose name matches the property string property, the function returns zero.
     Otherwise, it returns a nonzero value suitable for use as the second argument
     to a subsequent call to towctrans.

SEE ALSO

     wctrans(3)

STANDARDS

     The towctrans() function conforms to ISO 9899: 1990 (``ISO C'').
