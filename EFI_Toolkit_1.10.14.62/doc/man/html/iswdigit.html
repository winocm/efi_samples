NAME

     iswdigit - decimal-digit wide character test

SYNOPSIS

     #include <ctype.h>

     int
     iswdigit(wint_t wc)

DESCRIPTION

     The iswdigit() function tests for any wide decimal-digit character.  In the
     ASCII character set, this includes the following characters:

     060 ``0''	   061 ``1''	 062 ``2''     063 ``3''     064 ``4''
     065 ``5''	   066 ``6''	 067 ``7''     070 ``8''     071 ``9''

RETURN VALUES

     The iswdigit() function returns zero if the wide character tests false and re-
     turns non-zero if the character tests true.
