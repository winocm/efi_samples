_LIBC_Cleanup 		     Libc    Manual 	       _LIBC_Cleanup

NAME

     _LIBC_Cleanup - perform libc cleanup operations.

SYNOPSIS

     #include <atk_libc.h>

     void
     _LIBC_Cleanup(VOID)

DESCRIPTION

     _LIBC_Cleanup() should be used by all EFI protocols and drivers that
     have called IntializeLibC() before they exit through unload.  This call
     will perform most cleanup operations associated with exit(3) but without
     calling the EFI Exit service call.  Cleanup operations include:

	   1.	Flush all open output streams.
	   2.	Close all open streams.
	   3.   Free all memory allocated through malloc(3), calloc(3),
	        and realloc(3).
