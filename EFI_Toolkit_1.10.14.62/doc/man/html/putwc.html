NAME

     fputwc, putwc, putwchar - output a wide character to a stream

SYNOPSIS

     #include <wchar.h>

     wint_t
     fputwc(wint_t c, FILE *stream)

     wint_t
     putwc(wint_t c, FILE *stream)

     wint_t
     putwchar(wint_t c)

DESCRIPTION

     The fputwc() function writes the wide character c (converted to an ``wchar_t'')
     to the output stream pointed to by stream.

     The putwc() macro acts essentially identically to fputwc(), but is a macro
     that expands in-line. It may evaluate stream more than once, so arguments
     given to putwc() should not be expressions with potential side effects.

     The putwchar() macro is identical to putwc() with an output stream of
     stdout.

RETURN VALUES

     The functions, fputwc(), putwc() and putwchar() return the wide character writ-
     ten.  If an error occurs, the value WEOF is returned.

SEE ALSO

     ferror(3),  wfopen(3),  getwc(3),  stdio(3)

STANDARDS

     The functions fputwc(), putwc(), and putwchar(), conform to ISO 9899: 1990
     (``ISO C'').
