NAME

     btowc - converts (unsigned char) c

SYNOPSIS

     #include <stdio.h>
     #include <wchar.h>

     wint_t
     btowc(int c)

DESCRIPTION

     The btowc() function determines whether c constitues a valid (one-byte)
multibyte character in the initial shift state.

RETURN VALUES

     The btowc() function returns WEOF if c has the value EOF or if (unsigned char)c does not
constitute a valid (one-byte) multibyte character in the initial shift state. Otherwise, it
returns the wide-character representation of that character.
     This function has not been actually implemented. So it will always return WEOF.

STANDARDS

     The btowc() function conforms to ISO 9899: 1990 (``ISO C'').
