NAME

     wmemcpy - copy wchar_t string

SYNOPSIS

     #include <string.h>

     void *
     wmemcpy(void *dst, const void *src, size_t len)

DESCRIPTION

     The wmemcpy() function copies len wchar_t's from string src to string dst.

RETURN VALUES

     The wmemcpy() function returns the original value of dst.

SEE ALSO

     bcopy(3),	memccpy(3),  memmove(3),  strcpy(3)

BUGS

     In this implementation wmemcpy() is implemented using bcopy(3),  and
     therefore the strings may overlap.  On other systems, copying overlapping
     strings may produce surprises.  A simpler solution is to not use
     wmemcpy().
