NAME

     iswspace - white-space wide character test

SYNOPSIS

     #include <wctype.h>

     int
     iswspace(wint_t wc)

DESCRIPTION

     The iswspace() function tests for the standard white-space wide characters.  In
     the ASCII character set, this includes the following characters:

     011 ht	   012 nl	 013 vt        014 np	     015 cr
     040 sp

RETURN VALUES

     The iswspace() function returns zero if the wide character tests false and re-
     turns non-zero if the wide character tests true.
